# A3 Rocket Consulting

Marketing site for A3 Rocket Consulting, built with TanStack Start and deployed on Netlify at [a3rocketconsulting.com](https://a3rocketconsulting.com).

## Tech Stack

- **Framework:** TanStack Start (React 19)
- **Routing:** TanStack Router (file-based)
- **Styling:** Tailwind CSS 4
- **Build:** Vite 7
- **Deployment:** Netlify

## Running Locally

```bash
npm install
npm run dev
```

The dev server starts at `http://localhost:3000`. Netlify CLI wraps it at port 8888 with full platform feature emulation.

## Building

```bash
npm run build
```

Output is written to `dist/client`.
